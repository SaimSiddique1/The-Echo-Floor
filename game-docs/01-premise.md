# 01 — Premise

## One Sentence

A solo-survival game set in an infinite skyscraper where every floor is almost identical to the last, except one detail is wrong, and you must find it — one wrong guess and your run is over.

## Genre

Solo-survival / observation.

Comparable to *How far can you go?* arcade games. Every player is in it for themselves. No teamwork, no voting, no discussion — just you, the floor, and the anomaly.

## Player Count

**2–8 players** per server. Each player has an individual run — one wrong guess eliminates you.

## Target Audience

- Roblox core demographic (9–16)
- Fans of liminal space / backrooms aesthetics
- Players who enjoy high-score chasing and personal bests
- Low barrier to entry: walk, look, pick room + type

## Why This Will Work

1. **Endless replayability** — how far can YOU go?
2. **Zero game knowledge required** — pick a room, pick a type, that's it.
3. **Liminal space aesthetic** is trending.
4. **Skill gap is visible** — leaderboard shows furthest floor reached.
5. **Low-end friendly** — small map, few objects, no AI.
6. **Monetization-ready** — elevator skins, player cosmetics, streak-freeze items.

## Core Loop

Lobby → elevator bay → floor N → search → lock in guess → intermission → reveal → correct? next floor. wrong? spectate.
